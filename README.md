# ftnt-sdw-training


## Initial config

- Connect to Bastion
- Change root passwd
  - If Bastion is Ubuntu 17.04
  - wget https://raw.githubusercontent.com/barbosm/ftnt-sdw-training/master/bastion_files/sources.list
  - mv sources.list /etc/apt/sources.list
  - apt update
- Execute bastion_files/prepare_bastion.sh
